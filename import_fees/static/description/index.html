Setup

HS Codes setup (added by this module)

This allows you to input various import rates and taxes. The feature is available in Inventory > Configuration > HS Codes

To enable customs fees calculation, the new Harmonized Code field should be defined either on Product or on Product Category hierarchy

Related setup in standard Odoo Inventory Module Related Odoo settings

For Landed Costs valuation to function properly, Product Categories need to have a Costing Method set on Average costs or FIFO

For foreign currencies to be enabled in Odoo, you need to enable it in Settings > Invoicing > Currencies Section > Multi Currencies
